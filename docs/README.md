## 📝 文档编写规范

### 文档命名
- 使用大写字母和下划线（如：`DEV_QUICK_START.md`）
- 或使用全小写和连字符（如：`ui-design-system.md`）
- 保持一致性

### 文档结构
1. 标题（H1）
2. 简介/概述
3. 目录（可选）
4. 主要内容（分章节）
5. 相关链接
6. 更新日期/版本信息

### 链接规范
- 使用相对路径
- 同目录：`./FILE.md`
- 上级目录：`../DIR/FILE.md`
- 根目录项目文件：`../../FILE`
